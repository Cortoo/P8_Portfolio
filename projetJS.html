Création d'un portfolio d'une architecte d'intérieur dynamique en JavaScript

Lien vers le repo Github : Repo Github


En partant d'une maquette figma existante, je devais dans ce projet développer et intégrer en JavaScript la page de présentation des travaux d'une architecte d'intérieur, la page de connexion de l'administrateur.trice du site, ainsi qu'une modale (et sa gestion) permettant d'uploader de nouveaux projets et médias. Ce projet m'a permis de comprendre le JavaScript et de l'apréhender. Le backend étant deja développer, je devais y faire appel afin de pouvoir enregistrer les travaux que l'administrateur.trice voulait uploader.

Grace à ce projet j'ai notamment pu me confronter a la gestion d'une page d'adminisatrateur, en mettant en place une page uniquement visible par celui ci. Malgré quelques difficultés dans la gestion des projets, en faisant appel a l'api deja mise en place, j'ai pu apprendre à en gérer le bon fonctionnement.
